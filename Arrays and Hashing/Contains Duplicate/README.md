# Neetcode - Contains Duplicate

## Problem
Given an integer array nums, return true if any value appears more than once in the array, otherwise return false.

## My Solution
I learned what a hashset is and used it to run efficently (O(n)) through the code with only 1 loop where the values in the set are being checked for double entires which return True If so, otherwise return False. Though in this case space is counted as O(n) due to the hashset, nevertheless still an efficent solution.
